# SubDuedMPGD
Subdued is a survival zombie horror WebGL Game.
# Objective:
Survive!
- Use your gold, weapons and environment to your advantage and hold off the zombie hordes.

# Default Controls:
- W,A,S,D - Move.
- Spacebar - Jump.
- Left Mouse Button - Shoot.
- E - To interact/buy.
- 1,2,3 - Use inventory items.
